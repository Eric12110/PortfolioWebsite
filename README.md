## ECPay 金流串接

1. 複製 `.env.example` 為 `.env`，並填入你的 ECPay 商店資料：
   ```bash
   cp .env.example .env
   ```
2. 更新以下必填變數：
   - `ECPAY_MERCHANT_ID`
   - `ECPAY_HASH_KEY`
   - `ECPAY_HASH_IV`
   - `ECPAY_RETURN_URL`（綠界付款完成後回呼的伺服器端網址）
3. 啟動伺服器：
   ```bash
   npm install
   npm start
   ```
4. 在瀏覽器開啟 `http://localhost:3000/payment.html`，填寫金額與聯絡資訊後即會跳轉至綠界測試金流。

> ⚠️ 預設使用綠界測試環境。若要串接正式環境請將 `.env` 中的 `ECPAY_OPERATION_MODE` 改為 `Production`。
